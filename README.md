# Subscription API

REST-сервис для учета подписок на Go.

## Как запустить
1. Сборка и запуск контейнера:
   ```bash
   docker-compose up --build
   ```
2. API будет доступно на `http://localhost:8080`.

## Документация
Спецификация Swagger находится в `docs/swagger.yaml`.
По адресу `http://localhost:8080/swagger/swagger.yaml`.
